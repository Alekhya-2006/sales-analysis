# Sales Analysis using Python

## Project Overview

This project analyzes the **Sample Superstore** dataset using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn** to explore business performance and generate meaningful insights from sales data.

The objective of this project is to perform **Exploratory Data Analysis (EDA)**, identify business trends, evaluate sales performance, and provide data-driven business recommendations.

---

## Dataset

**Dataset:** Sample Superstore

The dataset contains **9,994 sales transactions** and **21 columns**, including:

- Order ID
- Order Date
- Ship Date
- Customer Information
- Product Information
- Category
- Sub-Category
- Segment
- Region
- State
- Sales
- Profit
- Quantity
- Discount

## Key Findings

- Technology generated the highest sales and profit among all categories.
- Phones and Chairs were the highest revenue-generating sub-categories.
- Copiers generated the highest profit among all sub-categories.
- California recorded the highest sales, profit, and quantity sold.
- The West region generated the highest overall revenue and profit.
- The Consumer segment contributed the highest sales and quantity sold.
- Discounts above 30% negatively impacted profitability.
- Sales and profits showed consistent growth from 2014 to 2017.
- Sales peaked during September, November, and December.
- Higher sales generally resulted in higher profits, while excessive discounts reduced profitability.
